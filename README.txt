Чтобы запустить программу нужно активироать файл "hero.py"
если программа не запускается и выдает ошибку, проверьте наличие некоторых python библиотек, которые представлены в файле 
"requirements.txt", а если использовать в командной строке, перейдя на адрес директории с помощью cd, следующую команду:
$ pip install -r requirements.txt
- то все библиотеки установятся автоматически
